# OptionsTrader Documentation

This `docs/` folder contains all documentation for the OptionsTrader system.

## Structure

- `00-overview/` – Project goals, phases, and planning artifacts.
- `01-architecture/` – System-level diagrams and architecture descriptions.
- `02-domain-and-strategies/` – Trading domain assumptions and strategy rulebooks.
- `03-brokers-and-ingestion/` – Broker connectors (IBKR) and ingestion flows.
- `04-runtime-and-engine/` – Strategy runtime engine, context, and safety.
- `05-api-and-ui/` – External API and React UI (planned in future epics).
- `06-operations/` – Runbooks, environments, scheduled jobs.

Start with `00-overview/project_overview.md` and `01-architecture/system_architecture.md` if you're new.
